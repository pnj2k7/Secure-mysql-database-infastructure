# Secure-mysql-database-infastructure
DBMS_CW

Project Overview

This project demonstrates the design and implementation of a secure enterprise-level MySQL database infrastructure for a financial services organization.

The environment was implemented using:
CentOS 9
MySQL 8
MySQL Workbench
HAProxy

The project focuses on:
Secure database design
Authentication and access control
Role-Based Access Control (RBAC)
Threat mitigation
SQL Injection prevention
High availability architecture
MySQL replication
Audit logging
ISO 27001 compliance review


Security Features:
Strong authentication policies
Role-based privilege management
Password hardening
Audit logging
Replication security
Backup management
HAProxy load balancing
Least privilege principle
SQL Injection mitigation


System Architecture:
Architecture Components
Client Application
HAProxy Load Balancer
Primary MySQL Server
Read Replica MySQL Server
Backup System
Audit Logging System


Project Structure:
